遊戲程式設計

Making a game has always been a dream of mine. I love playing video games so I know how much hapiness a game can bring. I wanted to make a game that everyone would have fun playing it.

I started to learn how to make game last year. I choose Unity as my game engine because it has more tutorials and more people using it. In this way, not only more people can answer my questions, but also it is more convenient to solve bugs. I made a small 2D platformer last summer break. I realized that it is hard to make a interesting game by myself. It need so much time to make a single game.

In this semester, I have a class called game programming. The final project of this class is to make a game with our classmates, so I find a some like-minded friends to make a 3D action adventure game. The game we made is called "Find Your Way Back". It's a game that you need to control the robots character, which has three abilities, defeat the monsters and get out of the abandoned city. I'm in charge of how to control the movement of character, while other team members make different scenes and monsters.

Here is the demo video of our games. Hope you will like it. We are still improving, and we will continue to upload the latest progress if there is any.

做一款遊戲一直是我的夢想。喜歡打電動讓我知道一款遊戲能帶來多大的快樂。我想做一款每個人都能開心遊玩的遊戲

我從去年開始學習如何寫一款遊戲。我選擇Unity作為我的遊戲引擎是因為他是最多人使用也最多教學的遊戲引擎。這樣不僅比較多人能解答我的問題，遇到bugs時也比較方便解決。我在去年暑假時做了一款2D平台遊戲。我了解到光我一個人是不夠的。寫一款好玩的遊戲所需花費的時間與精力是多到無法想像的。

這學期，我有一門課叫做遊戲程式設計。這門課的final project是要與組員一起寫一款遊戲。我們寫了一個名為“Find Your Way Back”的遊戲。這是一款控制擁有三種能力的機器人角色打敗怪物並走出廢棄城市的遊戲。我負責寫如何操作腳色，而其他團隊成員製作不同的場景和怪物。
